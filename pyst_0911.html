f-string(Formatted String Literal) 개념 💡

f-string(Formatted String Literal)은 파이썬 3.6부터 도입된 문자열 포맷팅 방법 중 가장 간편하고 강력한 기능입니다. 문자열 앞에 `f` 또는 `F`를 붙이고, 중괄호 `{}` 안에 변수나 표현식을 직접 넣어주면 해당 값이 문자열 안에 삽입됩니다.

왜 f-string을 사용해야 할까요?

f-string 이전에는 `+` 연산자를 사용해 문자열을 이어붙이거나, `.format()` 메소드를 사용해야 했습니다. 하지만 이 방법들은 몇 가지 단점이 있었죠.

`+` 연산자

문자열이 아닌 다른 타입(예: 숫자)의 변수를 합칠 때 `str()` 함수로 일일이 변환해줘야 해서 번거롭고, 코드가 길고 가독성이 떨어집니다.

age = 18 print("제 나이는 " + str(age) + "살입니다.")
`.format()` 메소드

중괄호 `{}`의 순서를 맞춰야 하거나, 인덱스 번호를 명시해야 하는 번거로움이 있었습니다.

age = 18 print("제 나이는 {}살입니다.".format(age))

f-string은 이런 불편함을 해결합니다. 코드가 훨씬 직관적이고 간결해져서, 실수를 줄이고 코드를 읽기 쉽게 만들어줍니다.

f-string의 다양한 활용법 ✨

1. 변수 삽입

가장 기본적인 사용법입니다.

name = "철수" score = 95 print(f"학생 이름: {name}, 점수: {score}")
결과가 여기에 표시됩니다.
2. 간단한 연산

덧셈, 뺄셈, 곱셈 같은 간단한 수학 연산도 중괄호 안에서 바로 할 수 있습니다.

price = 1500 count = 3 print(f"총 가격은 {price * count}원입니다.")
결과가 여기에 표시됩니다.
3. 함수 호출

문자열을 다루는 `upper()` 함수나 다른 함수도 중괄호 안에 넣어 사용할 수 있습니다.

username = "admin" print(f"환영합니다, {username.upper()}님!")
결과가 여기에 표시됩니다.
4. 조건 표현식 (삼항 연산자)

간단한 if-else 문을 한 줄로 표현하는 조건 표현식도 f-string 안에서 사용할 수 있습니다.

score = 85 grade = "합격" if score >= 60 else "불합격" print(f"점수: {score}점, 결과: {grade}")
결과가 여기에 표시됩니다.
5. 정렬 및 포맷팅

숫자나 문자열을 특정 너비에 맞춰 정렬하거나 소수점 자릿수를 조절하는 등 포맷팅도 가능합니다. `:` 뒤에 포맷 코드를 추가합니다.

name = "파이썬" pi = 3.141592 print(f"이름: {name:<10}") # 10칸을 확보하고 왼쪽 정렬 print(f"원의 넓이: {pi:.2f}") # 소수점 둘째 자리까지 표시
결과가 여기에 표시됩니다.

능력치 분배 프로그램

stat_point = 10 # 기본으로 주어진 능력치 포인트 strength = int(input("힘(Strength)에 몇 포인트를 투자하시겠습니까? (최대 10): ")) stat_point = stat_point - strength if stat_point < 0: print("포인트가 부족합니다. 다시 시작하세요!") elif stat_point == 0: print("모든 포인트를 사용했습니다. 훌륭한 캐릭터네요!") else: agility = int(input(f"남은 {stat_point} 포인트를 민첩(Agility)에 투자하시겠습니까?: ")) stat_point = stat_point - agility if stat_point < 0: print("포인트가 부족합니다. 다시 시작하세요!") elif stat_point == 0: print("모든 포인트를 사용했습니다. 훌륭한 캐릭터네요!") else: intelligence = int(input(f"마지막 남은 {stat_point} 포인트를 지능(Intelligence)에 투자하시겠습니까?: ")) stat_point = stat_point - intelligence if stat_point < 0: print("포인트가 부족합니다. 다시 시작하세요!") else: print("캐릭터 생성이 완료되었습니다!") print(f"최종 능력치 - 힘: {strength}, 민첩: {agility}, 지능: {intelligence}")
