반복문의 개념 🔄

반복문(Loop)은 특정 코드 블록을 여러 번 실행해야 할 때 사용하는 프로그래밍 구조입니다. 반복문을 사용하면 코드를 효율적으로 줄이고 가독성을 높일 수 있습니다.

반복문은 크게 **`for` 반복문**과 **`while` 반복문** 두 가지 종류가 있습니다.

for 반복문 (For Loop)

`for` 반복문은 **정해진 횟수**만큼 반복하거나, 리스트나 튜플 같은 **반복 가능한(iterable) 객체**의 모든 요소를 하나씩 순회할 때 사용합니다.

기본 형식 (리스트 순회)
fruits = ["사과", "바나나", "딸기"] for fruit in fruits: print(f"제가 좋아하는 과일은 {fruit}입니다.")
결과가 여기에 표시됩니다.
횟수 지정 (`range()` 함수 사용)

특정 횟수만큼 반복하고 싶을 때는 `range()` 함수를 사용합니다. `range(5)`는 0부터 4까지의 숫자를 생성합니다.

# 5번 반복 for i in range(5): print(f"{i}번째 반복")
결과가 여기에 표시됩니다.

while 반복문 (While Loop)

`while` 반복문은 **조건이 참(True)인 동안** 코드 블록을 계속 반복 실행합니다. 반복 횟수가 불명확하거나 특정 조건이 충족될 때까지 실행해야 할 때 유용합니다.

기본 형식 (조건이 참일 때까지)
count = 0 while count < 3: print(f"현재 카운트: {count}") count = count + 1 # count를 증가시켜야 무한 루프에 빠지지 않습니다.
결과가 여기에 표시됩니다.

for문과 while문 비교 (1부터 5까지 출력)

두 반복문은 서로 다른 구조를 가집니다. **횟수가 명확**할 때는 `for`문이, **조건이 명확**할 때는 `while`문이 더 효율적입니다.

for문으로 작성

# 시작과 끝이 명확한 경우 for i in range(1, 6): # 1부터 5까지 print(i)
결과가 여기에 표시됩니다.

while문으로 작성

# 조건이 참인 동안 반복 i = 1 while i <= 5: print(i) i += 1 # 반복 조건 변화
결과가 여기에 표시됩니다.

반복 제어: break와 continue

반복문 내부에서 실행 흐름을 제어하는 두 가지 주요 명령어가 있습니다.

1. `break` 명령어: 반복문 즉시 종료

`break`는 **현재 실행 중인 반복문 전체**를 완전히 빠져나가게 합니다. 주로 특정 조건을 만족했을 때 더 이상의 반복이 필요 없을 경우 사용됩니다.

# i가 3이 되면 반복문 종료 for i in range(1, 6): if i == 4: print("반복 중지!") break print(f"현재 i: {i}")
결과가 여기에 표시됩니다.

2. `continue` 명령어: 현재 반복 건너뛰기

`continue`는 현재 진행 중인 반복(iteration)의 나머지 코드를 실행하지 않고, **다음 반복으로 바로 넘어갑니다.**

# i가 3일 때만 출력 건너뛰기 for i in range(1, 6): if i == 3: print("(3은 건너뜀)") continue print(f"현재 i: {i}")
결과가 여기에 표시됩니다.

3. `break`와 `continue` 함께 사용 (실전 예제)

하나의 반복문 내에서 `continue`로 특정 항목을 건너뛰고, `break`로 전체 반복을 조기에 종료하는 예시입니다. (던전 탐험 상황)

monsters = ["슬라임", "고블린", "트랩", "오크", "보스", "드래곤"] for monster in monsters: if monster == "트랩": print("트랩! 피해만 입고 통과.") continue if monster == "보스": print("보스 등장! 종료.") break print(f"{monster} 전투!")
결과가 여기에 표시됩니다.

중첩 `for` 반복문으로 구구단 출력하기

`for` 반복문을 중첩하여 2단부터 9단까지 구구단 전체를 출력하는 코드입니다. 코드를 실행하고 결과를 확인해 보세요.

for i in range(2, 10): # 2단부터 9단까지 반복 (i는 단) for j in range(1, 10): # 1부터 9까지 반복 (j는 곱하는 수) # f-string을 사용하여 결과를 깔끔하게 출력합니다. print(f"{i} x {j} = {i * j}") print("----------------") # 단이 끝날 때마다 구분선 출력
결과가 여기에 표시됩니다.

사용자 입력 기반: 1부터 N까지의 홀수 합 계산

사용자로부터 숫자를 입력받아 1부터 그 숫자까지의 홀수만을 더하는 프로그램입니다. `for`문과 `if` 조건문을 함께 사용하여 해결하는 예시 코드입니다. (웹페이지에서 실행이 안됨으로 파이썬 프로그램에서 직접 실행해보세요.)

N = int(input("숫자를 입력하시오 : ")) odd_sum = 0 # 1부터 N까지 반복 for i in range(1, N + 1): # 홀수인지 확인 (나머지가 0이 아니면 홀수) if i % 2 != 0: odd_sum += i print(f"1부터 {N}까지의 홀수의 합: {odd_sum}")
