파이썬 연산자 마스터하기

일상적인 계산부터 데이터 분석까지, 파이썬의 연산자가 어떻게 작동하는지 대화형 예제를 통해 직접 확인해보세요.

산술 연산자 (Arithmetic Operators)

수학 계산에 사용되는 연산자들입니다. 덧셈, 뺄셈, 곱셈, 나눗셈 등 익숙한 연산부터 몫과 나머지를 구하는 연산까지 살펴봅니다.

+ (덧셈)

두 수를 더하거나 문자열을 연결합니다.

print(10 + 5) print("Hello" + " World")
결과가 여기에 표시됩니다.

- (뺄셈), * (곱셈), / (나눗셈)

기본적인 사칙연산입니다. 나눗셈(/)은 항상 소수점 결과가 나옵니다.

print(10 - 3) print(4 * 5) print(20 / 4)
결과가 여기에 표시됩니다.

두 수를 나눈 정수 몫(Floor Division)과 나머지를 구합니다.

print(10 // 3) print(10 % 3)
결과가 여기에 표시됩니다.

수를 거듭제곱합니다.

print(2 ** 3)
결과가 여기에 표시됩니다.

비교 연산자 (Comparison Operators)

두 값의 관계를 비교하여 참(True) 또는 거짓(False)을 반환합니다. 이 결과는 조건문에서 매우 중요하게 사용됩니다.

두 값이 같은지 또는 다른지 비교합니다.

print(10 == 10) print(10 != 5) print("apple" == "Apple")
결과가 여기에 표시됩니다.

값의 크기를 비교합니다.

print(10 > 5) print(10 < 5) print(10 >= 10) print(5 <= 5)
결과가 여기에 표시됩니다.

학생 도전 과제: 직접 작성하고 실행하기

아래 코드는 웹페이지에서 직접 실행되지 않습니다. 파이썬 IDLE나 에디터에 직접 복사하여 입력 값을 넣고 결과를 확인해 보세요!

1. 랜덤 덧셈 프로그램

두 개의 무작위 숫자를 더하는 간단한 문제입니다. 답을 입력하고 맞는지 확인해 보세요.

# 이 프로그램은 파이썬 IDLE 또는 에디터에서 실행해야 합니다. # ----------------------------------------------- import random num1 = random.randint(1, 20) num2 = random.randint(1, 20) answer = int(input(f"{num1} + {num2} = ")) if answer == num1 + num2: print("정답입니다!") else: print("틀렸습니다. 정답은", num1 + num2, "입니다.") # -----------------------------------------------

2. 짝수/홀수 판별기

숫자를 입력받아 짝수인지 홀수인지 판별합니다. `%`(나머지 연산자)를 활용합니다.

# 이 프로그램은 파이썬 IDLE 또는 에디터에서 실행해야 합니다. # ----------------------------------------------- number = int(input("정수를 입력하세요: ")) if number % 2 == 0: print(f"{number}는 짝수입니다.") else: print(f"{number}는 홀수입니다.") # -----------------------------------------------

3. 함수를 활용하여 모양 그리기

함수를 정의하고, 인자를 바꿔가며 터틀의 색과 각도를 조절하여 원하는 모양을 그려보세요.

# 이 프로그램은 파이썬 IDLE 또는 에디터에서 실행해야 합니다. # ----------------------------------------------- import turtle # 터틀 객체 생성 t = turtle.Turtle() t.shape("turtle") t.speed(1) t.pensize(3) # 터틀의 색과 방향을 바꾸며 이동시키는 함수 정의 def draw_line(length, angle, color): t.color(color) t.right(angle) t.fd(length) # 함수를 여러 번 호출하여 사각형 그리기 print("터틀이 색을 바꾸며 사각형을 그립니다.") draw_line(100, 90, "red") draw_line(100, 90, "green") draw_line(100, 90, "blue") draw_line(100, 90, "purple") turtle.done() # -----------------------------------------------
