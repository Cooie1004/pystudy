Python 리스트(배열) 학습

리스트란? 인덱스로 접근하기 주요 함수 학생 도전 과제

Python 리스트(배열)의 모든 것

하나의 변수에 여러 개의 값을 담고 싶을 때 사용하는 자료구조입니다. 다양한 종류의 데이터를 순서대로 저장하고 관리할 수 있습니다.

인덱스로 데이터 접근하기

리스트에 저장된 각 값은 고유한 위치 번호, 즉 **인덱스**를 가집니다. 인덱스는 **`0`**부터 시작하며, 인덱스를 이용해 원하는 위치의 데이터에 접근하거나 수정할 수 있습니다.

인덱스 사용 예제

리스트 `fruits`에서 인덱스 `0`과 `3`의 값을 출력합니다.

fruits = ["사과", "바나나", "딸기", "포도"] print(fruits[0]) print(fruits[3])
결과가 여기에 표시됩니다.

음수 인덱스 사용 예제

음수 인덱스는 리스트의 뒤에서부터 접근합니다. `-1`은 마지막 원소입니다.

numbers = [10, 20, 30, 40] print(numbers[-1]) print(numbers[-2])
결과가 여기에 표시됩니다.

len() 함수 사용 예제

`len()` 함수는 리스트의 길이를 반환합니다.

my_list = ['a', 'b', 'c', 'd', 'e'] print(len(my_list))
결과가 여기에 표시됩니다.

del 키워드 사용 예제

`del` 키워드를 사용해 특정 인덱스의 요소를 삭제합니다.

my_list = ['사과', '바나나', '포도'] del my_list[1] print(my_list)
결과가 여기에 표시됩니다.

주요 함수와 예제

리스트의 요소를 추가, 삭제하거나 정렬하는 등 다양한 작업을 도와주는 유용한 함수들입니다.

len() 함수

리스트의 길이를 반환합니다.

fruits = ['사과', '바나나', '포도'] print(len(fruits))
결과가 여기에 표시됩니다.

append(값) 함수

리스트의 마지막에 값을 추가합니다.

fruits = ['사과', '바나나'] fruits.append('딸기') print(fruits)
결과가 여기에 표시됩니다.

sort() 함수

리스트의 요소를 오름차순으로 정렬합니다.

numbers = [3, 1, 2] numbers.sort() print(numbers)
결과가 여기에 표시됩니다.

reverse() 함수

리스트의 요소를 역순으로 정렬합니다.

numbers = [1, 2, 3] numbers.reverse() print(numbers)
결과가 여기에 표시됩니다.

index(값) 함수

특정 요소가 처음 나타나는 인덱스를 반환합니다.

my_list = ['a', 'b', 'c', 'b'] index_b = my_list.index('b') print(index_b)
결과가 여기에 표시됩니다.

insert(인덱스, 값) 함수

특정 인덱스에 값을 삽입합니다.

my_list = ["a", "c"] my_list.insert(1, "b") print(my_list)
결과가 여기에 표시됩니다.

remove(값) 함수

리스트에서 첫 번째로 발견된 값을 삭제합니다.

my_list = [1, 2, 3, 2] my_list.remove(2) print(my_list)
결과가 여기에 표시됩니다.

pop() 함수

특정 인덱스의 요소를 삭제하고 반환합니다. 인덱스 생략 시 마지막 요소가 삭제됩니다.

my_list = [10, 20, 30] item = my_list.pop(1) print(item) print(my_list)
결과가 여기에 표시됩니다.

count(값) 함수

리스트 내 특정 요소의 개수를 세어 반환합니다.

my_list = [1, 2, 2, 3] count_2 = my_list.count(2) print(count_2)
결과가 여기에 표시됩니다.

학생 도전 과제: 직접 실행하고 분석하기

아래 코드를 실행해 보고, 리스트의 최종 상태를 분석해 보세요. 이 코드는 리스트의 요소를 삭제하고, 추가하고, 정렬하는 과정을 한 번에 보여줍니다.

성적 관리 프로그램

scores = [85, 92, 78, 92, 65] print("1. 초기 점수:", scores) # 78점 삭제하기 scores.remove(78) print("2. 78점 삭제 후:", scores) # 100점 추가하기 scores.append(100) print("3. 100점 추가 후:", scores) # 점수를 오름차순으로 정렬하기 scores.sort() print("4. 점수 정렬 후:", scores) # 92점이 몇 개인지 세기 count_92 = scores.count(92) print("5. 92점의 개수:", count_92)
결과가 여기에 표시됩니다.
